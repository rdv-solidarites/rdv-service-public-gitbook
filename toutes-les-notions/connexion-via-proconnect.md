# Connexion via ProConnect

Afin de renforcer la sécurité de notre produit et l'accès aux données des usagers, tous les comptes agents étant liés à [ProConnect](https://www.proconnect.gouv.fr/) devront obligatoirement utiliser ce moyen de connexion à partir du 16 février 2026.

{% hint style="info" %}
Si vous avez des comptes dans votre organisation qui n’utilisent pas ProConnect (par exemple, un agent qui n’a pas ProConnect, un intervenant, un compte guichet, etc.) ceux-ci ne sont pas impactés par cette modification.
{% endhint %}

Dans le but de pouvoir vous accompagner au mieux dans ce changement, **nous vous invitons dès maintenant à ne plus utiliser votre mot de passe pour vous connecter**. Lors de vos prochaines connexions à RDV Service Public/RDV Solidarités, utilisez le bouton « Se connecter avec ProConnect ».

Si toute fois, vous rencontrez des difficultés pour vous connecter par ce moyen, notre équipe est à votre disposition pour aider via la [page de contact](https://rdv.anct.gouv.fr/aide/demande_support/new?role=agent\&sujet=Connexion+ProConnect).
